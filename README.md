# 🧞 ChintaBot — জাদুকরী এআই এর মনের খেলা

**ChintaBot** হলো বাংলাদেশের প্রথম এবং সর্বাধুনিক এআই-চালিত বাংলা ক্যারেক্টার গেসিং গেম। এটি আপনার মনের গভীরে লুকিয়ে থাকা যেকোনো জনপ্রিয় চরিত্র বা ব্যক্তিকে তার জাদুকরী লজিক এবং বুদ্ধিদীপ্ত চালের মাধ্যমে নিখুঁতভাবে চিনে ফেলতে পারে! এটি আধুনিক কৃত্রিম বুদ্ধিমত্তা এবং নান্দনিক ডিজাইনের এক অনন্য সৃজনশীল মেলবন্ধন।

![ChintaBot Preview](public/og-image.png)

## ✨ কেন ChintaBot ব্যতিক্রমী? (Features)
- 🧞 **জাদুকরী জিনী (Genie):** `Google Gemini 2.1 Flash` চালিত আমাদের জিনী এখন আরও স্মার্ট। এটি স্বয়ংক্রিয়ভাবে মডেল সুইচ করে আপনাকে বিরামহীন গেম খেলার অভিজ্ঞতা দেয়।
- 👥 **গ্লোবাল মাল্টিপ্লেয়ার:** `MongoDB Atlas` এর মাধ্যমে এখন বন্ধুদের সাথে রুম-কোড শেয়ার করে লাইভ প্রতিযোগিতা করার সুবিধা। 
- 🎮 **চ্যালেঞ্জ এর মাত্রা (Difficulty Levels):**
  - 🟢 **সহজ:** ১০০টি প্রশ্ন এবং ৩টি ইঙ্গিত — দীর্ঘ এবং মজার সফর।
  - 🟡 **মিডিয়াম:** ৫০টি প্রশ্ন এবং ১টি ইঙ্গিত — ভারসাম্যপূর্ণ চ্যালেঞ্জ।
  - 🔴 **কঠিন:** মাত্র ৩০টি প্রশ্ন — জিনীর ধারালো বুদ্ধিমত্তার আসল পরীক্ষা!
- 💎 **গ্লাস-মর্ফিজম ডিজাইন:** প্রিমিয়াম ফ্রস্টেড গ্লাস ইউআই এবং মনোমুগ্ধকর ডাইনামিক ব্যাকগ্রাউন্ড। 
- 🏆 **অ্যাচিভমেন্ট ও পরিসংখ্যান:** আপনার প্রতিটি জয় এবং হার এখন ডাটাবেসে চিরস্থায়ীভাবে সংরক্ষিত থাকবে।

## 🛠 টেকনিক্যাল আর্কিটেকচার (Tech Stack)
- **Framework:** Next.js 16.1.7 (Latest)
- **Styling:** Tailwind CSS v4.0 (Modern Engine)
- **Database:** MongoDB Atlas (Cloud Database)
- **AI Engine:** Google Generative AI (Multi-model Fallback)
- **State Management:** React 19 Context & Hooks

## 🚀 গেমটি আপনার কম্পিউটারে শুরু করার নিয়ম
১. রিপোজিটরি ক্লোন করুন:
   ```bash
   git clone https://github.com/ta-syn/chintabot.git
   cd chintabot
   ```
২. প্রয়োজনীয় ফাইল ইনস্টল করুন:
   ```bash
   npm install
   ```
৩. পরিবেশ সেটআপ (Environment Setup):
   প্রজেক্টের রুট ফোল্ডারে একটি `.env.local` ফাইল তৈরি করে নিচের তথ্যগুলো দিন:
   ```env
   GEMINI_API_KEY=আপনার_গুগল_এআই_কি
   MONGODB_URI=আপনার_মঙ্গোডিবি_এটলাস_লিংক
   ```
৪. গেমটি চালু করুন:
   ```bash
   npm run dev
   ```
